Algorithm Order food and drinks
1.	Start
2.	Showing the food and beverage menu
3.	Input Add food / drink
4.	Input order quantity
5.	Add another order?
    •	If yes, then the previously selected food will be added to the order cart, then return to choosing food/drinks
    •	If no, then input delivery location
6.	Calculate total price (price food/drinks * order quantity)
7.	Show total price
8.	Input payment (online pay / cash on delivery)
9.	Order ready to ship
10.	End
